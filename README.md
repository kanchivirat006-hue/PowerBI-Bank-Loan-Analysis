
📊 Bank Loan Analysis Dashboard | Power BI
Project Overview

Developed an interactive Bank Loan Analysis Dashboard in Power BI to analyze lending performance, borrower behavior, loan quality, and financial metrics. The dashboard provides actionable insights into loan applications, funded amounts, repayments, interest rates, and debt-to-income ratios, helping stakeholders make data-driven decisions.

Key Features

✅ Loan Portfolio Performance Monitoring

✅ Good Loan vs Bad Loan Analysis

✅ Month-over-Month (MoM) & Month-to-Date (MTD) Tracking

✅ Regional Lending Analysis

✅ Loan Purpose & Term Analysis

✅ Borrower Employment & Home Ownership Insights

✅ Interactive Dashboard Navigation
